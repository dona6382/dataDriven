# DATA-DRIVEN
배치로 데이터를 수집, 변환, 저장할 수 있는 어플리케이션과 데이터를 제공해주는 API 서버

## 목적
- 대용량 데이터 처리

## 요구사항
- Node.js (버전)
- TypeScript (버전)

## 데이터 정의
- productReview
- productReviews
- productOverview

#### ProductDetails의 정보를 활용하여 ProductReviews를 조회하고, ProductDetails와 ProductReviews를 합하여 ProductOverview를 구성합니다.

## 진행상황
- 진행중
